# Toko Kelontong Pak Cik & Aimar

## Deskripsi
Aplikasi web untuk mengelola inventori produk menggunakan ExpressJS, jQuery, dan Bootstrap.

## Fitur
- CRUD Produk
- DataTable sederhana
- Modal konfirmasi delete
- Penyimpanan JSON

## Cara Menjalankan
1. npm install
2. node app.js
3. buka http://localhost:3000

## Teknologi
- ExpressJS
- jQuery
- Bootstrap
