1. Siapkan Bahan-bahannya (Install Dependencies)
Pertama, pastikan kamu sudah menginstal Python di komputer. Kalau sudah, kita perlu mengunduh beberapa "perpustakaan" kode agar aplikasinya bisa berjalan sempurna.

Buka terminal atau command prompt, lalu ketik perintah ini:

Bash
pip install -r requirements.txt
2. Jalankan Aplikasinya
Setelah semua bahan siap, sekarang waktunya menghidupkan aplikasi. Masih di terminal yang sama, jalankan file utamanya dengan perintah:

Bash
python app.py
Tunggu sebentar sampai muncul tulisan yang menandakan server lokal sudah aktif.

3. Buka di Browser
Nah, sekarang aplikasinya sudah "bernyawa"! Untuk melihat tampilannya, buka browser kesayangan kamu (Chrome, Edge, atau Firefox), lalu ketik alamat berikut di kolom pencarian:

http://127.0.0.1:5000
